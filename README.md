# Coding-Puzzles

All things coding related and you want to share with your colleagues.
No matter if LeetCode, AdventOfCode, Codinggame, Codecomabt, or any other coding platform.

## Folder structure:
    ```
    └── PLATFORM
        ├── SECTION
        ├── ...
        │   └── TASK
        │       ├── README.txt  // riddle description (copy&paste)
        │       └── USER        // your github name
    ```

### Sample
- AdventOfCode
    ```
    └── adventofcode              // PLATFORM
        ├── 2024                  // SECTION year
        │   └── 01                // TASK day
        │       ├── README.txt    // riddle description (copy&paste)
        │       └── booya2nd      // your github name
        │           ├── input.txt // riddle input (different for almost everyone)
        │           ├── part1.js  // part1 solution (there is always a second part)
        │           └── part2.js  // part2 solution
    ```
